-----------

^^^^

**event info dict**

+----------+----------------------------------------------+--------------------------------------------------------+
| dict key | type                                         | description                                            |

| key      | slice, index (int) or numpy-like fancy index | key at which vertex positions data were indexed/sliced |
+----------+----------------------------------------------+--------------------------------------------------------+
| value    | int | float | array-like                     | new data values for points that were changed           |
+----------+----------------------------------------------+--------------------------------------------------------+
